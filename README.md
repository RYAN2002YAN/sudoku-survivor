# Sudoku Survivor 🧩👹

**实时地牢逃亡 × 数独解谜** — 一边 WASD 躲避怪物，一边解 6×6 数独。

## 玩法

| 操作 | 效果 |
|------|------|
| 完成**一行** | 身后生成冰墙 ❄️（挡怪 5 秒） |
| 完成**一列** | 最近怪物眩晕 ⚡（定身 3 秒） |
| 完成**一宫** (2×3) | 全部怪物冰冻 🧊（4 秒） |
| 填入**错误数字** | 怪物速度翻倍 🔥（3 秒） |
| 解完整板 | 通关，进下一关 🎉 |

## 控制

- **WASD** 移动 / **鼠标点击** 数独格子 / **数字键 1-6** 填入 / **空格** 开始
- 右上角 🌐 切换中英文 / 🔊 切换音效+BGM

## 难度

**简单 / 中等 / 困难 / 压力** — 影响怪物速度、数独空格数、地图墙壁密度、初始生命

## 技术

纯 HTML + CSS + JS，零依赖，零构建工具。AI 合成音效和自适应 BGM（Web Audio API）。

## 运行

浏览器直接打开 `index.html`，或用任意静态服务器：

```bash
python3 -m http.server 8080
```
